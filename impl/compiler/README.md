# Compiler

This directory will contain the LinxISA compiler components (e.g. LLVM backend, assembler integration, ABI/calling
convention, and tests).

All encode/decode and instruction definitions must reference `spec/isa/spec/current/linxisa-v0.3.json`.

Canonical compiler bring-up plan and status: `impl/compiler/COMPILER_PLAN.md`.
